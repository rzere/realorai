# Security Policy

## Supported Versions

We currently support the latest commit on the `main` branch.

## Reporting a Vulnerability

- Please do not open public GitHub issues for security vulnerabilities.
- Instead, report via GitHub Security Advisories (preferred) or contact the maintainers privately.
  - From your fork, go to "Security" → "Advisories" → "Report a vulnerability".
  - If Security Advisories are unavailable, open a private discussion and we will follow up.

## Disclosure Policy

1. We will acknowledge the report within 72 hours.
2. We will investigate, develop a fix, and coordinate a disclosure timeline.
3. We will credit reporters if desired.

## Scope

This repository and its code. Third-party services (e.g., Hugging Face Inference, OpenAI APIs) are out of scope; please report issues affecting those services to their respective providers.
